player view going crazy when chunks are reloading

Cewein 2020-2-19

The player view is moving at unwanted places when chunks are reloaded . this is might be due to the mouse not being restricted to the game windows.

a possible fix can be done with :

private void Start()
{
     Cursor.lockState = CursorLockMode.Locked;
     Cursor.visible = false;
}

void Update()
{
    if (Input.GetKeyDown(KeyCode.Escape))
    {
        Cursor.lockState = CursorLockMode.None;
        Cursor.visible = true;
    }
}
